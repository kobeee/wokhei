【游戏名称】：Wok Hei Master (或者 The Toss)
【一句话Pitch】：感受“中国功夫烹饪”的火焰与节奏，别把炒饭撒在地上。

【核心玩法循环 (Core Loop)】：
1. 备菜：各种颜色的方块（代表米饭、鸡蛋、葱花、牛肉）掉进锅里。
2. 颠勺 (The Toss)：
   - 鼠标左键按住 = 压锅蓄力。
   - 鼠标上划松开 = 颠起。
   - 鼠标移动 = 接住。
3. 控火 (Fire Control)：
   - 食材在空中停留时间越长，冷却越快。
   - 接触锅底时间越长，温度越高（有进度条，变红会焦）。
   - 完美状态：食材在火苗上掠过，触发“Wok Hei (镬气)”特效（金光一闪 + 巨大的 *WHOOSH* 声）。
4. 上菜：倒进盘子，评分基于：色泽（没焦）、完整度（没撒）、镬气值。

【文化“翻译”策略 (Westernization)】：
1. Uncle Roger 风格：
   - 游戏中有一个 NPC 师傅（穿白色背心的大爷）。
   - 你失败了他会骂：“Haiyaa! You killed the rice!”
   - 你成功了他会说：“Fuiyoh!”（这种梗在西方互联网极具传播力）。
2. 食物反馈：
   - 重点不是食谱，是声音。
   - 必须录制真实的铁锅撞击声（Clang! Clang!）和油脂爆裂声。这是 ASMR 玩家的最爱。

【独立开发者 MVP 范围】：
- 只需要做一个场景：灶台。
- 只需要一口锅，一种物理引擎。
- 这是典型的“小游戏”，非常适合发在 r/WebGames 或 Itch.io 上测试手感。
- 如果手感好，可以直接上 Steam 卖 $2.99。
